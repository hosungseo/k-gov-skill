# Examples

`k-gov-skill`의 runnable examples입니다. 에이전트가 읽기 쉬운 JSON index는 [`examples/index.json`](index.json)에 있습니다.

## 바로 실행

```bash
npm run example:public-data
npm run example:gazette
npm run example:org-quota
npm run example:hwp
npm run example:law
npm run example:prepare-api
```

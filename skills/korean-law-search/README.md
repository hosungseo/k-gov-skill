# korean-law-search

## 목적

법령/조문/행정규칙/판례·해석례 후보를 찾아 공무원 업무 문서에 붙일 수 있는 근거표로 정리하는 스킬입니다.

## 사용자 신청/로그인

선택.

공개 검색 경로와 기존 코퍼스로 기본 탐색을 할 수 있습니다. 공식 API를 안정적으로 호출하려면 사용자가 법제처 국가법령정보 API OC 값을 준비하는 편이 좋습니다.

## 필요한 환경변수

```bash
LAW_GO_KR_OC="..." # 선택
```

## 신청처

- https://www.law.go.kr
- 국가법령정보 공동활용 신청/OC 발급

## 사용 예시 프롬프트

```text
정부조직법에서 행정안전부 소관 사무와 조직 설치 근거를 찾아줘.
현행 조문 중심으로 인용 후보를 만들고, 부칙보다 본문 조문을 우선해줘.
```

## 예상 산출물

- 법령명
- 조문번호
- 조문 제목/본문 발췌
- 관련성 설명
- 원문 링크 또는 API 호출 정보
- 불확실한 부분
- 업무 문서에 붙일 수 있는 인용문안

## 키 없을 때 fallback

- 공개 웹 검색/수동 링크 후보 제시
- 이미 구축된 법령/업무 노트가 있으면 해당 코퍼스 우선 탐색
- API 호출이 필요한 경우 OC 발급 방법 안내

## 주의

- 법률 자문처럼 단정하지 않습니다.
- 가능하면 현행 조문 본문을 우선합니다.
- 부칙, 개정연혁, 시행일은 별도 확인 항목으로 분리합니다.

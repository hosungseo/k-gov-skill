# national-assembly-search

## 목적

의안, 회의록, 국회 일정, 의원실 질의 대응 자료를 탐색하는 스킬입니다.

## 사용자 신청/로그인

필요/선택.

열린국회 API 실호출에는 API key가 필요할 수 있습니다. 키가 없으면 공개 웹/샘플/fallback으로 후보를 정리합니다.

## 필요한 환경변수

```bash
ASSEMBLY_API_KEY="..."
```

## 신청처

- 열린국회정보 API 포털
- 검색어: 의안정보, 회의록, 국회일정, 위원회 회의

## 사용 예시 프롬프트

```text
최근 정부조직법 관련 의안과 회의록 후보를 찾아줘.
의안명, 제안자, 진행상태, 관련 조문, 우리 업무와 연결되는 쟁점을 정리해줘.
```

## 예상 산출물

- 의안/회의록 후보
- 발의자/위원회/일자
- 진행상태
- 관련 법령/정책 연결
- 국회 대응 쟁점
- 답변 초안에 쓸 근거

## 키 없을 때 fallback

- API key 신청 안내
- 공개 웹 검색 후보
- 기존 `korean-government-api-bundle` 문서/샘플 활용

## 주의

- 국회자료는 최신성과 상태값이 중요합니다.
- 의원실 대응 문안은 최종 제출 전 사람 검토가 필요합니다.

# hwp-document-tools

HWP/HWPX 문서를 Markdown/JSON으로 읽고, 표·항목·쟁점을 추출해 에이전트가 검토할 수 있게 만드는 스킬입니다.

## 사용자 신청/로그인

불필요.

로컬 파일을 대상으로 동작합니다. 기관 내부문서는 외부 업로드 없이 로컬에서 처리하는 것을 원칙으로 합니다.

## 필요한 환경변수

없음.

## 사용 예시 프롬프트

```text
이 HWPX 검토자료를 Markdown으로 변환하고, 요구사항·근거자료·쟁점·보완요청 후보를 표로 정리해줘.
원문 구조는 최대한 보존해줘.
```

## 예상 산출물

- Markdown 변환본
- 표 목록
- 주요 항목/요구사항
- 쟁점표
- 누락/불명확한 부분
- 검토의견 초안 재료

## 키 없을 때 fallback

키가 필요 없습니다. 다만 HWP binary 품질에 따라 `hwp5txt`, `kordoc`, `rhwp`, HWPX XML 추출 등 여러 경로를 시도합니다.

## 주의

- 비공개 문서는 외부 서비스에 업로드하지 않습니다.
- 변환 텍스트는 표 병합/셀 순서 오류가 있을 수 있어 중요 표는 원문 대조가 필요합니다.

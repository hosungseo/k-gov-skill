# Use Cases

## 바이브코딩 공무원이 바로 써볼 만한 일

### 공공데이터 API 호출 준비

공공데이터포털에서 API를 찾고 활용신청해야 하는지 확인한 뒤, serviceKey 환경변수와 호출 파라미터를 정리합니다. 키가 없으면 실패하지 않고 신청 방법과 fallback을 안내합니다.

### 법령 근거 붙인 검토 문안 만들기

법령명을 던져 요약을 받는 대신, 조문번호·원문 인용·시행일·출처·확인일이 있는 citation packet으로 정리합니다.

### 관보에서 직제·정원 관련 문서 찾기

관보 PDF를 하나씩 열지 않고, 직제·정원·조직개편·입법예고 키워드로 문서 후보를 만들고 원문 대조 필요 지점을 남깁니다.

### HWP/HWPX 검토자료 구조화

HWP/HWPX를 로컬에서 다루는 전제로 표, 요구사항, 쟁점, 보완요청 후보를 분리합니다. 비공개 문서 외부 업로드 금지가 기본 원칙입니다.

### 직제·정원 요구자료 검토

요구자료를 바로 판단하지 않고, 요청 요약, evidence plan, 쟁점표, 보완요구, 반대검토, 검토의견 초안으로 나눕니다.

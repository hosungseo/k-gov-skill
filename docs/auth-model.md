# 인증/키 모델

`k-gov-skill`은 k-skill식 구동방식을 따른다. 스킬은 제공하지만, 공공 API 권한이나 개인 로그인은 사용자가 직접 준비한다.

## 권한 유형

| 유형 | 의미 | 예시 |
|---|---|---|
| `none` | 키 없이 공개 데이터/문서로 동작 | 관보 Markdown, 보도자료 Markdown |
| `user-key-required` | 사용자가 직접 활용신청/API key 제공 필요 | 공공데이터포털 API |
| `optional-user-key` | 키가 있으면 공식 API, 없으면 샘플/fallback | 법령/국회 일부 경로 |
| `manual-login` | 사용자가 직접 로그인/다운로드/승인 필요 | 기관 내부시스템, 일부 원문 다운로드 |
| `hosted-fallback` | 제한된 데모용 proxy/sample 사용 | 공개 데모 |

## 스킬 README 필수 항목

- 사용자 신청/로그인
- 필요한 환경변수
- 신청처/발급처
- 키 없을 때 동작
- 예시 프롬프트
- 예상 산출물
- 주의사항

## 원칙

공공데이터포털 serviceKey가 필요한 작업을 운영자 키로 몰래 대체하지 않는다. 사용자가 키를 갖고 있으면 공식 API를 호출하고, 없으면 신청 방법과 fallback을 안내한다.

# Contributing

`k-gov-skill`은 한국 공공업무에 실제로 도움이 되는 작은 스킬과 레시피를 모읍니다.

## 좋은 기여

- 공무원 업무에서 반복되는 작은 문제를 해결한다.
- 필요한 API key, 로그인, 활용신청 여부를 숨기지 않는다.
- 공식 출처와 기준일을 남긴다.
- 키가 없을 때 fallback 또는 신청 안내를 제공한다.
- 개인정보·민감정보·처분 판단 경계를 명확히 적는다.

## 새 스킬 체크리스트

- `skills/<skill-name>/README.md` 작성
- 사용자 신청/로그인: 불필요 / 선택 / 필요 중 명시
- 필요한 환경변수 명시
- 예시 프롬프트 포함
- 예상 산출물 포함
- 키 없을 때 fallback 포함
- 안전 주의 포함

## 원칙

공공 API key나 개인 인증정보를 커밋하지 마세요.
